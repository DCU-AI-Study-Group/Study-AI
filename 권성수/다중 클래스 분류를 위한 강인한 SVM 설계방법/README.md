## 발표자 후기

이진 분류에 주로 사용되는 Support Vector Machine을 다중 분류에도 사용되는 점을 확실하게 이해하는 시간이 되었던 것 같습니다. SVM을 활용한 다중 분류에는 다양한 문제점들이 존재하는데 그러한 문제점들을 해결할 수 있는 다양한 방법들이 있음을 확인하고 이러한 방법들을 연구해 볼 수도 있을 것 같다는 생각을 하게 되어 좋은 경험이 되었습니다.
